Política de Privacidade

Última atualização: 20 de junho de 2026

O ClipFlow é uma ferramenta de automação de conteúdo que utiliza a API oficial do TikTok (Content Posting API) para publicar vídeos em nome de criadores autorizados.

1. Dados que coletamos

O ClipFlow não coleta, armazena ou compartilha dados pessoais de usuários finais. Os únicos dados processados são:

  • Token de acesso OAuth 2.0: gerado pelo TikTok após autorização explícita do criador, armazenado localmente e usado exclusivamente para autenticar chamadas à API de publicação.
  • Arquivos de vídeo: enviados diretamente aos servidores do TikTok via upload chunked (FILE_UPLOAD). Nenhum vídeo é retido após a confirmação de publicação.
  • Metadados da postagem: título/descrição fornecidos pelo criador, enviados junto com o vídeo na chamada de init.

2. Como usamos os dados

Todos os dados são usados exclusivamente para:

  • Autenticar o criador junto à API do TikTok (OAuth 2.0)
  • Iniciar, fazer upload e confirmar a publicação de vídeos na conta TikTok do criador autorizado
  • Consultar informações do criador (avatar, username) para exibição na interface de confirmação pré-postagem

Nenhum dado é utilizado para publicidade, análise de terceiros, treinamento de IA ou qualquer outra finalidade.

3. TikTok API e Cookies

O ClipFlow integra-se com a API oficial do TikTok (Content Posting API). O TikTok pode utilizar cookies e tecnologias de rastreamento próprias durante o fluxo de autorização OAuth. Consulte a Política de Privacidade do TikTok para detalhes.

O ClipFlow não utiliza cookies, pixels de rastreamento ou qualquer tecnologia de fingerprinting própria.

4. Armazenamento e segurança

  • Tokens OAuth são armazenados em arquivo local criptografado no dispositivo do criador.
  • Nenhum dado é enviado a servidores de terceiros além dos endpoints oficiais da API do TikTok (open.tiktokapis.com e open-upload.tiktokapis.com).
  • Vídeos são transmitidos diretamente ao TikTok — não há armazenamento intermediário em nuvem.

5. Compartilhamento de dados

O ClipFlow não vende, aluga ou compartilha dados com terceiros. A única entidade que recebe dados é o TikTok, como parte necessária da funcionalidade de publicação.

6. Consentimento e revogação

O uso da API requer autorização explícita do criador via OAuth 2.0. O criador pode revogar o acesso a qualquer momento:

  • Pelo painel de apps autorizados do TikTok (Configurações → Segurança → Apps Autorizados)
